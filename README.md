# RankMaster

This is a Next.js application built with Firebase Studio that allows you to rank products within categories.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI**: [React](https://react.dev/) with [ShadCN UI](https://ui.shadcn.com/) components
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Backend & Database**: [Firebase](https://firebase.google.com/) (Authentication and Firestore)
- **Generative AI**: [Genkit](https://firebase.google.com/docs/genkit)

This app uses Firebase for all backend services, including data persistence with Firestore and user authentication.
