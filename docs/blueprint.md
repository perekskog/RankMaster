# **App Name**: RankMaster

## Core Features:

- Category Management: Add, edit, and delete product categories. Only empty categories can be deleted.
- Product Ranking (Graded): Rank products within a category using a 3, 5, or 7-point grading system.
- Product Ranking (Comparative): Rank products by selecting two products and indicating which is better.
- Combined Ranking Display: Display products within a category, sorted by a ranking algorithm that combines all ranking methods used.
- Product Addition: Add products with a text description and an optional picture.
- Product Deletion: Delete products from a category.
- Persistent Storage: Products and categories are stored using an external persistent mechanism such as a SQL Database in a dotnet backend written in c#.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent clarity and reliability.
- Background color: Light grey (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Yellow-orange (#F9A825) to highlight important actions and provide visual interest.
- Font: 'Inter' sans-serif for a modern, objective feel. Good for both headlines and body text.
- Use simple, consistent icons to represent categories and ranking actions.
- Clean and intuitive layout, with clear hierarchy and spacing to ensure ease of use.
- Subtle animations on product ranking updates and category changes for user feedback.
